# Focus - Pomodoro Timer

Focus é uma aplicação de timer Pomodoro criada como projeto de aprendizado para desenvolver habilidades em JavaScript através de um produto real e funcional.

A ideia foi simples: ao invés de fazer exercícios isolados, construir algo completo — uma ferramenta de produtividade com ranking global que recompensa os usuários que mais estudam.

---

## Sobre o Projeto

Focus é uma aplicação web full-stack que combina a técnica Pomodoro com um sistema de ranking competitivo. Os usuários podem acompanhar suas sessões de foco, competir com outros com base no tempo total de estudo e personalizar as configurações do timer conforme sua rotina.

O projeto foi desenvolvido individualmente como forma prática de aprender JavaScript, React e desenvolvimento web moderno.

---

## Funcionalidades

- Timer Pomodoro com modos de foco, pausa curta e pausa longa
- Ranking global baseado no tempo total de foco
- Autenticação completa com cadastro, login e recuperação de senha
- Histórico de sessões com filtros por semana, mês e ano
- Configurações de timer personalizadas por usuário
- Sistema de feedback para sugestões e relato de bugs
